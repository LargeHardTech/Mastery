# Mastery README
## 一.软件介绍
Mastery是一个简单易用的学习辅助工具，旨在帮助用户更高效地学习和掌握新知识。它提供了多种功能，包括：
- 1.数学计算练习：Mastery提供了各种数学计算练习，帮助用户提高数学技能和计算能力。
- 2.英语单词记忆：Mastery提供了英语单词记忆功能，帮助用户扩大词汇量，提高英语水平。
Mastery使用.NET 8.0 Winforms开发，支持Windows操作系统，低配电脑也能流畅运行。
